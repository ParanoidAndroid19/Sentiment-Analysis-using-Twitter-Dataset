# Sentiment Analysis
Analyses tweets containing a particular keyword using Python & NLP and categorises them as positive, strongly positive, weakly positive, neutral, negative, weakly negative and strongly negative.

# Requirements

1. Python 3.6
2. Matplotlib
3. Tweepy
4. Textblob

# Run

1. After installing all the dependencies run main.py
2. Enter the keyword to search and the number of tweets to analyse.
